For a long time, I was thinking about a tool that will provide a set of CSS custom properties with the mouse position, scroll position, random values etc. And since I didn't find anything that I would like, I decided to write the tool myself. I called it "Medium" because it connects CSS and JS worlds. The library itself is quite simple, but I used it for a few demos on CodePen and it really speeds things up. Now it's <a href='https://github.com/sfi0zy/medium'>available on GitHub</a> and as <a href='https://www.npmjs.com/package/css.medium.js'>NPM package</a>.
