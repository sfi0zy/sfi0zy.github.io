# Personal website

Hi! I'm Ivan Bogachev and this is the repository for my personal website - sfi0zy.github.io

## Development

```sh
git clone git@github.com:sfi0zy/sfi0zy.github.io.git
cd sfi0zy.github.io
make install
make watch
```

## Deployment

```sh
make deploy
```

Or, if you need to rebuild the *_site* only:

```sh
make build
```
