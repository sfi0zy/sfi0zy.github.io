# Personal blog

Hi! I'm Ivan Bogachev and this is a repository for [my personal blog](https://sfi0zy.github.io).

It includes:

- Online version of the [evolutionary theory of behavior](https://sfi0zy.github.io/evolutionary-theory-of-behavior/) with latest fixes and comments.
- Related [thoughts](https://sfi0zy.github.io/tag/evolution/) about human behavior, psychiatry, philosophy, and how computer science can be applied to all of that.
- Archive of old posts with links to my articles about creative front-end development, animations on CodePen, and other IT-related stuff.

## Development

```sh
git clone git@github.com:sfi0zy/sfi0zy.github.io.git
cd sfi0zy.github.io
npm i
npm run lint
npm run build
npm run serve
```

## Deployment

```sh
./deploy.sh
```
