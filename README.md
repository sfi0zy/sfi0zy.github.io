# Personal website

Hi! I'm Ivan Bogachev and this is the repository for my personal website - sfi0zy.github.io

## Development

```sh
git clone git@github.com:sfi0zy/sfi0zy.github.io.git
cd sfi0zy.github.io
npm i
npm run lint
npm run build
npm run serve
```

## Deployment

```sh
./deploy.sh
```
